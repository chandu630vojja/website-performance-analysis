# Website Performance Analysis Tool

A comprehensive web application for analyzing website performance metrics, including Core Web Vitals, resource optimization, SEO recommendations, and beautiful performance visualizations.

## Features

- **Real-time Performance Analysis**: Analyze any website URL instantly
- **Core Web Vitals**: LCP, FID, CLS, TTFB, and FCP measurements with thresholds
- **10+ Performance Graphs**:
  - Core Web Vitals Bar Chart
  - Performance Score Gauge
  - Resource Distribution Pie Chart
  - Resource Size Breakdown
  - Time Series Trend Analysis
  - Radar Chart for Metrics
  - Improvement Opportunities
  - Normalized Score Comparison
  - Metric Breakdown with Thresholds
  - Recommendation Priority Analysis
- **Resource Analysis**: Count and analyze JS, CSS, and image resources
- **SEO Checks**: Meta tags, heading structure, and image alt attributes
- **Performance Scoring**: 0-100 performance score with detailed breakdown
- **Recommendations**: Actionable optimization suggestions with priority levels
- **PDF Reports**: Generate and download detailed performance reports with graphs
- **Batch Analysis**: Analyze up to 10 websites simultaneously
- **Website Comparison**: Compare 2-5 websites side by side
